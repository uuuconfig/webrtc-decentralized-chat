# P2P Chat 用户指南

## 删除群聊

### 步骤1：选择要删除的群聊
在左侧群聊列表中点击要删除的群聊，使其显示深色背景（表示已选中）。

### 步骤2：打开群成员面板
选中群聊后，右侧会自动显示"群成员"面板，显示该群的所有成员及其连接状态。

### 步骤3：点击删除按钮
在群成员面板的顶部，有两个按钮：
- **删除按钮**（垃圾桶图标）- 左边
- **关闭按钮**（×）- 右边

点击**删除按钮**（垃圾桶图标）。

### 步骤4：确认删除
系统会弹出确认对话框，显示：
```
确定要删除群聊 [群ID前8位] 吗？
```

点击**确认**按钮完成删除。

### 删除后的效果
- ✅ 群聊从左侧列表中消失
- ✅ 群聊的所有历史消息被删除
- ✅ 如果正在查看该群，会自动切换到"未选择聊天"状态
- ✅ 显示提示信息："群聊已删除"

## 查看群成员

### 方法1：自动显示（推荐）
1. 在左侧群聊列表中**点击任意群聊**
2. 右侧会自动显示"群成员"面板
3. 面板中列出该群的所有成员

### 方法2：手动打开
如果群成员面板被关闭了，可以：
1. 确保已选中一个群聊
2. 右侧会自动显示群成员面板

### 群成员面板显示的信息

#### 群链接ID
- 显示群的唯一标识符（前16位 + ...）
- 可点击**复制按钮**复制完整ID分享给其他人

#### 成员列表
每个成员显示以下信息：

| 信息 | 说明 |
|------|------|
| 头像 | 成员名字的首字母 |
| 成员名字 | 成员的昵称 |
| IP地址 | 成员的网络地址（如果已连接） |
| 连接状态 | 显示该成员的P2P连接状态 |

#### 连接状态说明
- 🟢 **P2P 已连接** - 与该成员的P2P连接已建立，可以直接通信
- 🟡 **连接中...** - 正在建立与该成员的P2P连接
- 🔴 **未连接** - 与该成员的P2P连接未建立
- 🟢 **（我）** - 表示当前用户自己

### 群成员面板的操作

#### 关闭面板
点击群成员面板右上角的**×按钮**可以关闭面板。
- 关闭后，聊天区域会变宽
- 再次点击群聊会重新打开面板

#### 删除群聊
点击群成员面板右上角的**删除按钮**（垃圾桶图标）可以删除整个群聊。

#### 复制群链接ID
点击群链接ID旁边的**复制按钮**，可以复制完整的群ID到剪贴板，分享给其他人加入。

## 常见问题

### Q: 删除群聊后能恢复吗？
**A:** 不能。删除是永久操作，会删除该群的所有本地历史消息。如果需要重新加入，需要获取群链接ID。

### Q: 群成员面板为什么没有显示？
**A:** 
1. 确保已选中一个群聊（左侧列表中显示深色背景）
2. 如果面板被关闭了，再次点击群聊会重新打开
3. 检查屏幕宽度是否足够显示面板

### Q: 为什么看不到某个成员的IP地址？
**A:** 
- 如果成员未连接（显示"未连接"状态），则不会显示IP地址
- 只有已建立P2P连接的成员才会显示IP地址

### Q: 删除群聊会通知其他成员吗？
**A:** 会。删除群聊时，系统会向信令服务器发送离开消息，其他群成员会收到"[你的名字] 离开了群"的通知。

### Q: 如何重新加入已删除的群聊？
**A:** 
1. 获取群链接ID
2. 在左侧"群聊"面板中，找到"加入群（粘贴群链接 ID）"输入框
3. 粘贴群链接ID
4. 点击加入按钮

## 快速操作总结

| 操作 | 步骤 |
|------|------|
| **查看群成员** | 点击左侧群聊列表中的群 → 右侧自动显示群成员面板 |
| **删除群聊** | 选中群聊 → 点击群成员面板的删除按钮 → 确认删除 |
| **复制群链接** | 选中群聊 → 点击群成员面板的复制按钮 → 粘贴分享 |
| **关闭群成员面板** | 点击群成员面板右上角的×按钮 |
| **重新打开群成员面板** | 再次点击左侧群聊列表中的群 |
